The Master Theorem for Solving Recurrence Relations

The master theorem:

Given a recurrence relation in the form:

T(n)=aT(nb)+f(n)T(n) = aT(\frac{n}{b}) + f(n)T(n)=aT(bn​)+f(n)

where

f(n)∈Θ(nk)f(n) \in \Theta(n^{k})f(n)∈Θ(nk)

Case 1 - If

a<bka < b^{k}a<bk

T(n)∈Θ(nk)T(n) \in \Theta(n^{k})T(n)∈Θ(nk)

T(n)∈Θ(nklog⁡(n))T(n) \in \Theta(n^{k}\log(n))T(n)∈Θ(nklog(n))

a>bka > b^{k}a>bk
